## ▶️ How to Run

1. Clone the repo:
   ```bash
   git clone https://github.com/JanviNandre/data_structures_in_popular_apps/edit/main/README.md
   cd data-structures-demo
   ```

2. Compile & run:
   ```bash
   javac *.java
   java Main
   ```

## 🎯 Key Features
